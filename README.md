# Reconnect (wagwan)

A warm, playful iOS app to help you stay in touch with the friends who matter most.

## Features

- **Friend Management**: Add friends you want to stay connected with
- **Smart Reminders**: Set custom reminder schedules for each friend
- **Contact Tracking**: Log when you last reached out
- **Status Indicators**: See at a glance who's overdue, due soon, or on track
- **Calendar Integration**: Creates calendar events that deep-link back to the app

## Tech Stack

- **SwiftUI** - Modern declarative UI
- **SwiftData** - Persistence with automatic CloudKit sync capability
- **EventKit** - Calendar integration for reminders

## Design Philosophy

Reconnect uses a warm, inviting design language:
- **Colors**: Coral, sage, lavender, sunflower
- **Typography**: Rounded, friendly fonts
- **Interactions**: Bounce animations and haptic feedback
- **Inspiration**: Dot, Streaks, Duolingo

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Getting Started

1. Clone the repository
2. Open `Reconnect.xcodeproj` in Xcode
3. Build and run on simulator or device
